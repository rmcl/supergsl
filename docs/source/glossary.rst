##############################################
Glossary
##############################################

.. glossary::

    fGSL
        The original Genome Specification Language written in F# programming language.

    Primer
        A short nucleic acid sequence that provides a starting point for DNA polymerases.
        For the purposes of synthetic biology, primer almost always refers to "Synthetic Primers"
        used to initiate PCR reactions (https://en.wikipedia.org/wiki/Primer_(molecular_biology)#Uses_of_synthetic_primers).

    Paired Primer
      A :term:`Paired Primer` is :term:`Primer` with two regions `body` and `tail` which are complementary to different parts. These primers
      are used as a "homology bridge" to connect two parts during a PCR reaction.

    Sublime Text
      Sublime Text is a sophisticated text editor for code, markup and prose. You'll love the slick user interface, extraordinary features and amazing performance.
